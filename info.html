Event Information

CODE IT

Code it is a competition where the participants will have to write correct and efficient codes for the given set of problem statements. This is an online coding round in which the winning team will get a direct entry in the final round. Qualifying teams will then have to code for the problems of higher difficulty level in a limited time.

Rules and Regulations-
1. The groups will be given maximum 2 hours to complete.
2. Permitted languages will be C, C++, Python and Java.
3. A group of two is allowed or you can participate individually.

RECODE

Recode is a competitive coding competition based on perfection along with speed !
In the first half of this event, mcq questions will be asked. Maximum time provided will be 30 minutes. In the second half, a set of executable files will be given.The participants can run the files any number of times they want. Maximum time for scondd half is 1 hour. From the given custom inputs by the participant they need to figure out the logic of the code of the executable file and code it.
Rules and Regulations-
1. Languages allowed are C, C++, Python and Java.
2. The participants will be allowed to use computer to code the solution if required or they can solve it on paper.
